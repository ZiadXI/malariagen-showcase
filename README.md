# MalariaGEN Natural-Language Interface (Mockup)

A front-end UI concept demonstrating a Natural-Language Interface (NLI) for querying MalariaGEN genomic datasets. 

##  Overview
This project is a static prototype created to showcase a proposed user experience for genomic data analysis. It visualizes a workflow where a user inputs a natural language prompt, and the system outputs:
1. The generated Python query code.
2. A responsive data visualization (stacked bar chart).
3. A natural language summary of the analysis.

##  Live Demo
**[Click here to view the live interactive mockup](https://ziadxi.github.io/malariagen-showcase/)**


##  Tools used
* **HTML5:** Semantic structure and layout.
* **CSS3:** Custom styling, Flexbox layout, and a completely custom-built pure CSS stacked bar chart (no external charting libraries used).

##  Key Features
* **Data Visualization:** Pure CSS stacked bar chart illustrating kdr (Vgsc-995F) mutation frequencies across *An. gambiae*, *An. coluzzii*, and *An. arabiensis*.
* **Code Display:** Styled mock-execution environment for Python data querying.
* **Responsive Layout:** Clean, split-panel interface optimized for modern desktop viewing.
